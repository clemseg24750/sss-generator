# SSS Generator

App Node.js locale pour générer des vidéos de release et visuels podcast.

## Prérequis

- **Node.js** ≥ 18
- **FFmpeg** installé et accessible dans le PATH

### Installer FFmpeg

**macOS** (Homebrew) :
```
brew install ffmpeg
```

**Windows** : télécharger depuis https://ffmpeg.org/download.html et ajouter le dossier `bin/` au PATH.

**Linux** :
```
sudo apt install ffmpeg
```

## Lancer l'app

```
npm install
npm start
```

Ouvrir **http://localhost:3000**
